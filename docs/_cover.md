# Meadow Graph Client <small>1</small>

> Pull comprehensions of data from relational database graphs by filter, not by JOIN

- Load a meadow schema, get an auto-wired join graph for free
- Query by `{Entity, Filter}` with dot-notation cross-entity references
- Weighted path scoring with hints and manual-path overrides
- Pluggable data-request transport for HTTP, IPC, or test fakes
- First-class Fable service provider
